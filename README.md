# Environment variables
## Howto 
We use `dontenv` to manage environment variables, please follow instruction below:
1. Create `.env` file in the root of this project.
2. Paste variables below with its value in the file.

## Variables
```
DB_USER=
DB_PASS=
DB_NAME=
DB_HOST=
DB_PORT=
TOKEN_SECRET=
PASSWORD_SECRET=
```
